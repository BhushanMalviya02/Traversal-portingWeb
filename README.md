# 🚚 Traversal – Moving Service Website

A responsive and modern **moving service website** built using **HTML, CSS, and Bootstrap 5**.
This project showcases a clean UI with reusable Bootstrap components like navbar, hero section, features, carousel, and footer.

---

## 📌 Project Overview

**Traversal** is a startup-style landing page designed for a moving service company.
The goal of this project is to create a visually appealing and user-friendly interface using Bootstrap.

This project demonstrates:

* Responsive design
* Bootstrap components usage
* Clean layout structure
* Basic frontend development skills

---

## 🖼️ Features

* 🔹 Responsive Navigation Bar
* 🔹 Hero Section with CTA buttons
* 🔹 Features Section (Professional, Countrywide, Personal Touch)
* 🔹 Image Carousel
* 🔹 Footer with multiple sections
* 🔹 Mobile-friendly layout

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Bootstrap 5**

---

## 💡 Key Learnings

* Understanding Bootstrap grid system
* Building responsive layouts
* Using components like navbar, carousel, and cards
* Structuring a real-world landing page
